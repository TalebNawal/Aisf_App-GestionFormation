# Application de Gestion des Sessions de Formation des Employés

Cette application de gestion des sessions de formation des employés a été conçue et développée pour simplifier le processus de planification, de suivi et de communication autour des sessions de formation au sein d'une organisation. Elle intègre également un système de messagerie automatisée pour faciliter la communication avec les participants.

## Outils Utilisés

**Langage de Programmation** : Java

**Framework d'Interface Graphique** : JavaFX

**Modélisation** : UML

**Gestion de Projet** : JIRA

**Contrôle de Version** : Git

**Conception Graphique** : Scene Builder

**Base de données** : MySQL

## Fonctionnalités Clés

**Gestion des Sessions de Formation** : Permet aux administrateurs de créer, planifier et organiser des sessions de formation en spécifiant les dates, les lieux, les instructeurs et les sujets.

**Inscription des Employés** : Les employés peuvent s'inscrire aux sessions de formation qui les intéressent, ce qui facilite la gestion des inscriptions et des effectifs.

**Suivi de la Progression** : Permet de suivre la progression de chaque employé inscrit aux sessions de formation, générant ainsi des rapports pour évaluer leur développement.

**Messagerie Automatisée** : Intègre un système de messagerie automatisée pour envoyer des rappels, des notifications et des informations importantes aux participants et aux instructeurs.

**Gestion de Projet** : Utilise JIRA pour gérer les tâches liées au développement de l'application.

## Acteurs

- Administrateur
  - S’authentifier
  - Récupérer mot de passe
  - Gérer les assistants
  - Gérer les formations
  - Lister les employées
  - Affecter des formations à des agents
  - Gérer la présence
  - Rechercher les formations  et les agents
  - envoyer les emails d’invitation
  - envoyer les emails de présence
  - Afficher les listes des agents, formations et présence
  - Afficher statistique
  - Se déconnecter


- Assistant de présence
  - S’authentifier
  - Consulter les listes de présence
  - Rechercher la présence par agent ou par formation
  - Rechercher une formation
  - Afficher la liste des agents pour chaque formation
  - Rechercher un agent
  - Marquer la présence
  - Se déconnecter



## Comment Utiliser l'Application

**Configuration de l'Environnement** : Assurez-vous que vous avez Java et JavaFX installés sur votre système.

**Clonage du Projet** : Clonez ce dépôt GitHub sur votre machine en utilisant la commande Git suivante :

```git clone https://github.com/votre-nom/utilisateur/formation-management-app.git```

**Exécution de l'Application** : Naviguez vers le répertoire cloné et exécutez l'application en utilisant JavaFX.

**Utilisation** : Suivez les instructions de l'interface utilisateur pour créer des sessions de formation, gérer les inscriptions des employés, suivre la progression, et utiliser les fonctionnalités de messagerie automatisée.
